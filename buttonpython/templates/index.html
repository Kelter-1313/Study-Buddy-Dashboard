Zoom Link Sharing is a tool that allows users to send a Zoom link to their study partners' emails all from one place.

The Pomodoro Timer provides users with a time management tool that encourages them to work through large projects in an efficient method. The Pomodoro Method consists of 4 25-minute work intervals seperated by five minute breaks.

The Study Tips tool provides users with randomized tips that may help them with everyday school activities. NOTE : Only works on windows computers at the moment.

Quick Translate is a tool that allows users to input foreign text (in Chinese, Spanish, Portuguese, and English), the language that text is in, and the language they want to translate that text to, all while never having to leave the page.

WikiSpeedia provides users with the ability to input a topic they would like to learn more about, and displays a short summary of that topic from Wikipedia.
